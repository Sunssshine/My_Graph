Построение графа при помощи списка инцидентности.

В поле, доступном для ввода, находящемся правее большого поля с надписью "GRAPH" возможно ввести свой граф.

Правила представления графа при построении:
- Все пробелы на этапе парсинга строки будут проигнорированы, так что их количество не ограничено.
- Первая буква каждой строки - название точки, из которой будет проведено ребро в последующие точки строки.
- Затем обязан следовать символ ':'
- Затем следуют буквы, обозначающие названия точек, связанных с первой точкой, если необходимо указать
  вес ребра проведенного в эту точку из первой точки, после точки вводится число обозначающее вес ребра.

Пример ввода графа в текстовом виде:

B : A101 K102 L103
A : M2 D34
K : A30 L20001
L :
M : B4 K21
D :

Рассмотрим первую строку, она означает, что:
1). Будет создана точка с названием B
2). Будет создана точка А, затем из точки B будет проведено направленное ребро в точку А с весом "101"
3). Будет создана точка К, затем из точки B будет проведено направленное ребро в точку К с весом "102"
4). Будет создана точка L, затем из точки В будет проведено направленное ребро в точку L с весом "103"

Обратите внимание, если вы не зададите вес ребрам точек посредством ввода числа после её имени,
её вес будет выбран по умолчанию, и будет равен нулю.

Для построения графа необходимо нажать на кнопку со знаком "*", если всё сделано правильно, на сцене слева
появится граф построенный на основе вашего ввода.

Также, если граф был введен некорректно, построения не произойдет, но вам будет сообщено место ошибки.
