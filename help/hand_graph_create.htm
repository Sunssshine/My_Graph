Построение графа "вручную".

1)  Двойной клик по полю, на котором при запуске находится надпись "GRAPH" вызовет окно ввода названия точки.
    - Название точки должно содержать только буквы, быть не пустым, а также не совпадать с названиями других точек.
    - После ввода названия, на сцене будет создана точка с этим именем.

2)  Нажатие правой кнопкой мыши по существующей на сцене точке вызовет контекстное меню, которое позволит:
    2.1) Нажатие на кнопку "Rename" вызовет окно переименования точки, правила переименования описаны выше.
    2.2) Нажатие на кнопку "Connect" выделит точку особым цветом и приготовит её к соединению с другой точкой.
         Теперь нажмите правой кнопкой мыши на другую точку и снова выберите пункт "Connect", точки будут соединены,
         причём ребро будет направлено из первой выделенной точки во вторую.
    2.3) Нажатие на кнопку "Delete" удалит точку и все соединенные с ней ребра.

3. Нажатие правой кнопкой мыши на конец стрелки, указывающей на точку, вызовет контекстное меню, которое позволит:
    3.1) Нажатие на кнопку "Change weight" вызовет окно изменения веса ребра, в нём необходимо ввести желаемый вес ребра.
    3.2) Нажатие на кнопку "Delete direction" вызовет удаление этого направления, если ребро имело только одно направление -
         - оно будет удалено полностью, если же нет, будет удалена только та стрелка, из которой было вызвано контекстное меню.
