Контексное меню при нажатии на вершину правой кнопкой мыши

1)  Для cоединения вешины с другой нажмите Соединить с, после этого кликните по вершине, с которой хотите соединить.
	1.1)  Вызовите пкм контекстное меню вершины и нажмите Соединить с.
	1.2)  После этого появится стелка, которая будет следовать за курсором.
	1.3)  Кликните по вершине, к которой хотите присоединить.
	1.4)  Если ребро получается двунаправленным, то рисуется 2 дуги.
2)  Для переименоваяни вершины нажмите Переименовать.
3)  Для изменения цвета вершины нажмите Изменить цвет.
4)  Для запуска алгоритма поиска кратчайших путей в бесконтурном графе нажмите Поиск путей.
5)  Для удаления вершины нажмите Удалить вершину.
