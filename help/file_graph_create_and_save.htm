Построение графа из файла и сохранение графа в файл.

После того, как вы каким-либо из способов (см. п. 1 и 2) создали граф,
вы можете сохранить его в файл с расширением .gph, для этого необходимо нажать на кнопку "SAVE",
после чего выбрать путь и ввести желаемое название нового файла в появившемся окне.

Готово, теперь по выбранному вами пути находится файл с выбранным вами именем, доступный для открытия.

Чтобы открыть сохраненный ранее файл с расширением .gph необходимо нажать на кнопку "OPEN",
после выбрать желаемый файл с расширением .gph в выбранном вами расположении.
После того как вы утвердите файл как выбранный для открытия, на сцене будет построен граф.

Обратите внимание, граф сохраняется на основе текущего состояния на сцене, а не в поле ввода.
